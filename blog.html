Java Script workshops by WarsawJS

WarsawJS is a local community of JavaScript professionals and enthusiasts. They meet every month to talk and debate about their favorite programming language. This community is focused on sharing experience from different fields on which JS is used, from web browsers, to servers, dev tools, mobiles, smart tvs, micro controllers, databases, video games and so on.

I had the pleasure to attend Java Script workshops to broaden my programming knowledge in this language. There were many enthusiastic tutors such as @Bartosz Legięć, who managed my team, ready to help us with every problem we find in our project. We spend the whole day coding, sharing experiences and solving problems. It was a huge inspiration for me!
